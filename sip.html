Custom Abilities

Creator:
Jordan Hall, Game Programming
Last Updated: 11/30/2020

Technical Field

This project involves the technical field of programming.

Background Information

I got my inspiration to create this project by watching anime and wanting to see something I thought would be interesting to put in a game.

Prior Art (legal term)

My inspiration does not come from a game that already exists, but an anime called “Death Marth to the Parallel World Rhapsody”. The anime is in a game like theme where the player/Character goes through the world and learns things by experiencing them. The pro for to video game is that it gives players more of a sense that there is infinite number of things they can do. A con to a system like this is that it can be overwhelming at first from all the things that can be done.

Project Description

My project will be a system that changes abilities based on the player’s actions. There will be event that player will be doing, and each event can affect the stats on the ability that the player might get. This will benefit the player because they can play the game in different ways than another player and different each time they play if played multiple times.

Innovation Claim

This project innovation is that events that are created can affect the ability stats of the player.
